Introduction

数据接收处理独立线程。

主要工作是将收到的数据进行解析并按MobileIMSDK框架的协议进行调度和处理。 本类是MobileIMSDK框架数据接收处理的唯一实现类，也是整个框架算法最为关 键的部分。

本线程的启停，目前属于MobileIMSDK算法的一部分，暂时无需也不建议由应用层自行调用。
